형용사의 부사형 익히기

학습목표: ~ないでください · 형용사의 부사형 (い형→く / な형→に)
부사형 만들기 규칙
い형용사 → く
大きい → 大きく / 早い → 早く / 短い → 短く ō-ki-i → ō-ki-ku / ha-ya-i → ha-ya-ku / mi-ji-ka-i → mi-ji-ka-ku 크다→크게 / 빠르다→빠르게 / 짧다→짧게
な형용사 → に
きれいだ → きれいに / 簡単だ → 簡単に / 真面目だ → 真面目に ki-re-i da → ki-re-i ni / kan-tan da → kan-tan ni / ma-ji-me da → ma-ji-me ni 깨끗하다→깨끗하게 / 간단하다→간단하게 / 성실하다→성실하게
~ないでください 연습 — 상황별
キム → 先生
窓を開けてもいいですか。 Ma-do o a-ke-te mo i-i de-su ka. 창문을 열어도 됩니까?
先生
寒いから、窓を開けないでください。 Sa-mu-i ka-ra, ma-do o a-ke-na-i-de ku-da-sai. 추우니까, 창문을 열지 말아 주세요.
キム → 先生
ここで遊んでもいいですか。 Ko-ko de a-son-de mo i-i de-su ka. 여기서 놀아도 됩니까?
先生
危ないから、ここで遊ばないでください。 A-bu-na-i ka-ra, ko-ko de a-so-ba-na-i-de ku-da-sai. 위험하니까, 여기서 놀지 말아 주세요.
キム → 先生
ごみを捨ててもいいですか。 Go-mi o su-te-te mo i-i de-su ka. 쓰레기를 버려도 됩니까?
先生
店の前ですから、ごみを捨てないでください。 Mi-se no ma-e de-su ka-ra, go-mi o su-te-na-i-de ku-da-sai. 가게 앞이니까, 쓰레기를 버리지 말아 주세요.
