▶ SESSION 15

て형 만들기 / てください

학습목표: ~がほしい · て형 만들기 · ~てください (~해 주세요)
て형 만들기 규칙
1그룹 く→いて / ぐ→いで
書く → 書いて / 泳ぐ → 泳いで ka-ku → ka-i-te / o-yo-gu → o-yo-i-de 쓰다→써서 / 수영하다→수영해서
1그룹 す→して
話す → 話して ha-na-su → ha-na-shi-te 이야기하다→이야기해서
1그룹 む·ぶ·ぬ→んで
飲む → 飲んで / 遊ぶ → 遊んで no-mu → non-de / a-so-bu → a-son-de 마시다→마셔서 / 놀다→놀아서
1그룹 る·う·つ→って
乗る → 乗って / 買う → 買って / 待つ → 待って no-ru → not-te / ka-u → kat-te / ma-tsu → mat-te 타다→타서 / 사다→사서 / 기다리다→기다려서
2그룹 る→て
食べる → 食べて / 見る → 見て ta-be-ru → ta-be-te / mi-ru → mi-te 먹다→먹어서 / 보다→봐서
3그룹 불규칙
する → して / 来る → 来て su-ru → shi-te / ku-ru → ki-te 하다→해서 / 오다→와서
회화 — 파티 초대
山田
今週の土曜日、時間がありますか。 Kon-shū no do-yō-bi, ji-kan ga a-ri-ma-su ka. 이번 주 토요일, 시간이 있습니까?
キム
はい、暇ですよ。どうしてですか。 Ha-i, hi-ma de-su yo. Dō-shi-te de-su ka. 네, 한가해요. 왜요?
山田
土曜日の３時から、私のうちで誕生日パーティーをします。キムさんも遊びに来てください。 Do-yō-bi no san-ji ka-ra, wa-ta-shi no u-chi de tan-jō-bi pā-tī o shi-ma-su. Ki-mu-san mo a-so-bi ni ki-te ku-da-sai. 토요일 3시부터 제 집에서 생일 파티를 합니다. 김 씨도 놀러 와 주세요.
山田
２号線の地下鉄に乗って、３つ目の駅で降りてくださいね。 Ni-gō-sen no chi-ka-te-tsu ni not-te, mit-tsu-me no e-ki de o-ri-te ku-da-sai ne. 2호선 지하철을 타고 3번째 역에서 내려 주세요.
山田
それから、駅で電話してください。 So-re-ka-ra, e-ki de den-wa shi-te ku-da-sai. 그리고 역에서 전화해 주세요.
キム
はい、わかりました。楽しみですね。 Ha-i, wa-ka-ri-ma-shi-ta. Ta-no-shi-mi de-su ne. 네, 알겠습니다. 기대되네요.
てください 요청 표현
まってくださいmat-te ku-da-sai기다려 주세요
みてくださいmi-te ku-da-sai봐 주세요
かいてくださいka-i-te ku-da-sai써 주세요
きてくださいki-te ku-da-sai와 주세요
はなしてくださいha-na-shi-te ku-da-sai말해 주세요
おりてくださいo-ri-te ku-da-sai내려 주세요
のってくださいnot-te ku-da-sai타 주세요
